## Phase 1 验收标准

### 必须通过 (Blocking)

- [ ] **构建成功**: `npm run build` 无错误
- [ ] **无重复入口**: ControlPanel 中无设置按钮
- [ ] **帮助功能**: 帮助按钮可正常打开/关闭
- [ ] **文档完整**: DEPRECATED.md、STORAGE_ARCHITECTURE.md 存在

### 应当通过 (Should)

- [ ] **构建警告**: chunk 警告不超过 1 个
- [ ] **工具函数**: courseDataUtils.js 导出所有函数
- [ ] **导入导出**: 导出的 JSON 可成功导入

### 可选通过 (Nice to have)

- [ ] **废弃清理**: 4 个废弃组件已删除
- [ ] **代码规范**: 无 ESLint 错误

## 测试环境

| 项目 | 版本/要求 |
|------|-----------|
| Node.js | ≥ 18.x |
| 浏览器 | Chrome/Edge 最新版 |
| 屏幕 | 1280x720 及以上 |

## 测试执行

```bash

npm install

# 2. 启动开发服务器
npm run dev

# 3. 手动执行测试用例

# 4. 验证构建
npm run build
```
