## 🎉 项目更新状态

## 🆕 新增功能

### 1. 默写练习模式

全新的独立练习模式，专注于听力+默写训练。

#### 核心特性

- ✅ **现代卡片式UI** - 类似 Duolingo/扇贝单词的视觉风格
- ✅ **单词块布局** - 句子拆分为独立的单词卡片
- ✅ **字符格子** - 每个字母在固定大小格子中显示
- ✅ **状态标签** - "正在输入..."、"✓ 完成"等提示

#### 交互改进

- ✅ **忽略大小写** - 输入 "t" 或 "T" 都算正确
- ✅ **自动跳过标点** - 逗号、句号等自动显示，无需输入
- ✅ **偷看答案** - 按 `Ctrl + ,` 快速查看完整句子

### 2. 模式切换

新增模式切换组件，可在两种模式间无缝切换：

- **普通练习模式** - 原有的终端式UI
- **默写练习模式** - 新的卡片式UI

## 🔧 技术实现

### 1. 大小写忽略

```javascript
// 比较时转换为小写
const isMatch = e.key.toLowerCase() === targetChar.toLowerCase();
```

### 2. 标点自动跳过

```javascript
const punctuationRegex = /[.,!?;:'"-]/;

// 检测标点并自动添加
if (punctuationRegex.test(targetChar)) {
  addChar(targetChar);
  return;
}
```

### 3. 偷看答案

```javascript
const isPeeking = useRef(false);

// 按住 Ctrl + , 显示答案
if (e.ctrlKey && e.key === ',') {
  e.preventDefault();
  isPeeking.current = true;
}

// 松开隐藏答案
window.addEventListener('keyup', (e) => {
  if (e.key === ',' && e.ctrlKey) {
    isPeeking.current = false;
  }
});
```

|------|------|

| 空格 | 输入空格（标点自动跳过）|
| Backspace | 删除字符 |
| Enter | 下一句 |
| Ctrl + , | 偷看答案（按住显示，松开隐藏）|

## 🎯 学习流程建议

### 默写模式学习流程

1. **听语音** - 系统自动播放句子语音
2. **开始默写** - 根据听到的内容输入
3. **卡住时** - 按 `Ctrl + ,` 快速偷看
4. **完成检查** - 完成后查看错误
5. **进入下一句** - 按 `Enter` 继续

### 最佳实践

- 📖 先听 1-2 遍，理解句子意思
- ✏️ 凭记忆默写，不要频繁偷看
- 🔄 完成后重听语音，对比检查
- 📊 记录错误单词，重点复习

. , ! ? ; : ' " -

- "the" 或 "THE" 或 "The" → ✅ 都算正确

### 3. 偷看答案

- **快速提示**：不用退出练习模式
- **即时隐藏**：松开按键立即恢复
- **无干扰**：不影响正常输入流程

## 🔄 模式切换逻辑

### 状态管理

```javascript
// Store 中的状态
practiceMode: 'normal' | 'dictation'

// 切换模式
togglePracticeMode: () => set((state) => ({
  practiceMode: state.practiceMode === 'normal' ? 'dictation' : 'normal'
}))
```

### 组件渲染

```javascript
{practiceMode === 'normal' ? (
  <>
    <ProgressBar />
    <TypingArea />
    <ControlPanel />
  </>
) : (
  <DictationMode />
)}
```
