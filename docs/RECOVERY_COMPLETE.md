## 🎮 功能演示

### 开始练习
1. 打开应用后自动播放第一句英文
2. 直接开始打字对应的字符
3. 已输入的字符显示为紫色（正确）或红色（错误）
4. 光标会自动移动到下一个字符

### 完成句子
1. 正确输入所有字符后显示 "✓ 完成！"
2. 按 `Enter` 进入下一句
3. 自动朗读新句子

### 操作快捷键
- **Enter** - 下一句
- **Space** - 输入空格
- **Backspace** - 删除字符
- **鼠标点击** 按钮可以：
  - 重新播放语音
  - 重新开始当前句子

## 🚀 进阶功能

### 添加音效（可选）

将 MP3 文件放入 `public/sounds/`：
- `success.mp3` - 成功音效
- `error.mp3` - 错误音效

注意：没有音效文件时应用仍然正常运行，只是没有声音反馈。

### 自定义主题

编辑 `src/index.css` 中的 `@theme` 块：

```css
@theme {
  --color-primary-500: #your-color;
  /* 修改其他颜色变量 */
}
```

## 💡 常见问题

**Q: 页面白屏？**
A: 按 F12 打开开发者工具，查看 Console 标签的错误信息。

**Q: 听不到语音？**
A: 确保浏览器支持 Web Speech API，并且没有静音。

**Q: 动画卡顿？**
A: 可以在组件中暂时移除 `motion` 组件，使用原生 HTML 标签。

**Q: 想要更多功能？**
A: 参考代码结构，在 `hooks/` 和 `components/` 中添加新功能。
