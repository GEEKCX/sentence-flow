# 🚀 默写练习模式快速使用指南

## ✨ 新功能速览

Sentence Flow 新增 **默写练习模式**，提供全新的卡片式UI和交互体验！

---

## 🎯 两种模式对比

| 特性 | 普通练习 | 默写练习 |
|------|----------|----------|
| UI风格 | 终端式 | 卡片式（Duolingo）|
| 大小写 | 严格匹配 | 自动忽略 |
| 标点符号 | 需要输入 | 自动跳过 |
| 偷看功能 | 无 | Ctrl + , |
| 视觉反馈 | 文本高亮 | 卡片状态 |
| 适用场景 | 精准练习 | 听力+默写 |

---

## 🎮 快速开始

### 1. 启动应用

```bash
cd sentence-flow
npm run dev
```

访问：http://localhost:5179

### 2. 选择模式

页面顶部有两个按钮：
- 🎧 **普通练习** - 原有的练习模式
- 📖 **默写练习** - 新的默写模式

点击即可切换

## 📖 默写模式操作指南

### 界面布局

```
┌────────┐  ┌────────┐  ┌────────┐
│  T  h  e  │  │  j  o  u  │  │  r  n  e  │
│正在输入... │  │          │  │          │
└────────┘  └────────┘  └────────┘

我正在跑步

[Enter 下一句] [Backspace 删除] [Ctrl + , 偷看答案]
```

### 基本操作

| 按键 | 功能 | 说明 |
|------|------|------|
| 字母键 | 输入字符 | 自动忽略大小写 |
| 空格 | 输入空格 | 标点自动跳过 |
| Backspace | 删除字符 | 删除最后一个字符 |
| Enter | 下一句 | 完成后进入下一句 |

### 高级功能

#### 偷看答案 - `Ctrl + ,`

**使用方法：**
1. 按住 `Ctrl + ,` - 显示完整句子
2. 松开按键 - 恢复隐藏状态

**使用场景：**
- 卡住时快速查看答案
- 检查拼写是否正确
- 学习新句子的完整形式

## 🎨 视觉反馈说明

### 单词卡片状态

#### 普通单词（未激活）
- 📦 白色背景
- 🔘 灰色边框
- ⬜ 字符格子为灰色

#### 激活单词（正在输入）
- 🟣 浅紫色背景
- 💜 紫色边框
- ✨ 当前字符闪烁
- 📝 显示"正在输入..."

#### 完成单词
- 🟢 浅绿色背景
- 💚 绿色边框
- ✅ 显示"✓ 完成"

### 字符格子状态

| 状态 | 背景色 | 文字色 |
|------|--------|--------|
| 未输入 | 浅灰 | 灰色 |
| 当前输入 | 浅灰（闪烁）| - |
| 正确输入 | 浅紫 | 紫色 |
| 错误输入 | 浅红 | 红色 |
| 标点符号 | 透明 | 灰色 |

## 💡 学习建议

### 默写模式学习流程

1. **听语音** - 系统自动播放句子语音
2. **理解意思** - 看中文翻译，理解句子含义
3. **开始默写** - 凭记忆输入英文句子
4. **偷看提示** - 卡住时按 `Ctrl + ,` 快速查看
5. **完成检查** - 完成后查看错误单词
6. **重听对比** - 按 `Ctrl + ,` 偷看，对比检查
7. **进入下一句** - 按 `Enter` 继续

### 最佳实践

- ✅ 先听 1-2 遍，理解句子
- ✅ 凭记忆默写，不要频繁偷看
- ✅ 完成后重听语音，对比检查
- ✅ 记录错误单词，重点复习
- ✅ 每天练习 10-15 个句子

## 🎯 常见问题

### Q: 为什么标点符号自动显示？

**A:** 默写模式专注于字母拼写，标点符号会自动处理，让你更专注于单词本身。

### Q: 偷看答案会记录吗？

**A:** 目前不会，但未来版本会添加偷看次数统计。

### Q: 可以设置难度吗？

**A:** 目前不支持，但计划在后续版本中添加（如显示首字母提示等）。

### Q: 两种模式的数据会分开记录吗？

**A:** 目前共享同一套句子数据，但练习进度独立。未来版本会支持独立统计。
