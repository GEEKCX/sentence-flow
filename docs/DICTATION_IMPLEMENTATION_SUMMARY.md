```
┌─────────────────┐
```

| 错误输入 | red-100 | red-500 |

## 🔧 技术实现

### 1. 大小写忽略

```javascript
// 比较时转换为小写
const isMatch = e.key.toLowerCase() === targetChar.toLowerCase();
```

**效果：**
- "The" 或 "the" 或 "THE" → ✅ 都算正确

### 2. 标点自动跳过

```javascript
const punctuationRegex = /[.,!?;:'"-]/;

if (punctuationRegex.test(targetChar)) {
  addChar(targetChar);  // 自动添加
  return;
}
```

**效果：**
- 标点符号自动显示为灰色
- 用户无需输入
- 光标自动跳到下一个字符

### 3. 偷看答案

```javascript
const isPeeking = useRef(false);

// Keydown - 显示
if (e.ctrlKey && e.key === ',') {
  e.preventDefault();
  isPeeking.current = true;
}

// Keyup - 隐藏
window.addEventListener('keyup', (e) => {
  if (e.key === ',' && e.ctrlKey) {
    isPeeking.current = false;
  }
});
```

**效果：**
- 按住 Ctrl + , 显示完整句子
- 松开立即隐藏
- 不影响正常输入

## 🎯 功能对比

### 普通模式 vs 默写模式

| 特性 | 普通练习 | 默写练习 |
|------|----------|----------|
| **UI风格** | 终端式 | 卡片式 |
| **大小写** | 严格匹配 | 自动忽略 |
| **标点符号** | 需要输入 | 自动跳过 |
| **偷看功能** | 无 | Ctrl + , |
| **视觉反馈** | 文本高亮 | 卡片状态 |
| **字符显示** | 单词内 | 独立格子 |
| **状态标签** | 无 | 有 |
| **适用场景** | 精准练习 | 听力+默写 |
