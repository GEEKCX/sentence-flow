## 如何诊断白屏问题：

### 1. 打开浏览器开发者工具

在浏览器中按 `F12` 或右键点击页面选择"检查"

### 2. 查看控制台（Console）标签

检查是否有以下类型的错误：

- **红色错误** - JavaScript 运行时错误
- **黄色警告** - 资源加载问题
- **紫色信息** - 我们添加的调试日志

### 3. 查看网络（Network）标签

检查资源是否正常加载：
- `/src/main.jsx` 应该加载成功
- `/src/index.css` 应该加载成功
- 所有 JS/CSS 资源状态应为 200

### 4. 检查元素（Elements）标签

- 找到 `<div id="root"></div>` 元素
- 查看是否有任何内容渲染在里面
- 检查是否有 CSS 样式应用

## 可能的问题和解决方案：

### 如果看到 "main.jsx loaded" 但没有其他日志：

问题：React 渲染失败

解决：检查 root 元素是否存在

### 如果看到 "Root element not found"：

问题：index.html 中没有 root 元素

解决：刷新页面

### 如果看到导入错误：

问题：模块路径不正确

解决：检查所有 import 语句

### 如果看到 "App rendered" 但页面仍然是白屏：

问题：CSS 没有加载或样式冲突

解决：检查 Tailwind 配置
